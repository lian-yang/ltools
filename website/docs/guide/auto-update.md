# 自动更新

LTools 内置自动更新机制，确保你始终使用最新版本。

## 更新机制

### 自动检查

- 应用启动 10 秒后自动检查更新
- 发现新版本时显示通知
- 点击通知可查看更新详情

### 手动检查

1. 打开「设置」→「关于」
2. 点击「检查更新」按钮
3. 查看检查结果

## 更新方式

### Windows

- 下载新版本安装程序
- 静默安装
- 自动重启应用

### macOS

- 下载新版本
- 显示下载完成提示
- 手动替换 Applications 中的旧版本

### Linux

- 下载新版本 (AppImage)
- 显示下载完成提示
- 手动替换旧版本文件

## 更新配置

目前自动更新行为是固定的，未来版本将支持：

- 关闭自动检查
- 设置检查频率
- 选择更新通道（稳定版/测试版）

## 更新日志

每次更新都包含详细的更新日志：

- 新功能介绍
- 问题修复
- 性能改进
- 安全更新

查看完整更新日志：[GitHub Releases](https://github.com/lian-yang/ltools/releases)

## 回滚版本

如果需要使用旧版本：

1. 访问 [GitHub Releases](https://github.com/lian-yang/ltools/releases)
2. 找到需要的版本
3. 下载对应平台的安装包
4. 安装旧版本

## 更新问题排查

### 更新检查失败

1. 检查网络连接
2. 检查是否使用代理
3. 尝试手动检查

### 下载失败

1. 检查磁盘空间
2. 尝试重新下载
3. 直接从 GitHub 下载

### 安装失败

- **Windows**: 关闭正在运行的 LTools 实例，重新安装
- **macOS**: 确保有足够的权限，尝试手动替换
- **Linux**: 确保 AppImage 有执行权限
