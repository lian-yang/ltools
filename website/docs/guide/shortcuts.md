# 快捷键

LTools 提供强大的快捷键系统，帮助你快速访问功能。

## 全局快捷键

这些快捷键在系统任何地方都有效：

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Cmd+5` (macOS) / `Ctrl+5` (Win/Linux) | 打开全局搜索 | 核心快捷键，快速唤起搜索窗口 |
| `Cmd+Shift+S` (macOS) / `Ctrl+Shift+S` (Win/Linux) | 截图 | 启动截图工具 |

## 搜索窗口快捷键

在全局搜索窗口中：

| 快捷键 | 功能 |
|--------|------|
| `↑` / `↓` | 上下选择结果 |
| `Enter` | 打开选中的插件/功能 |
| `Esc` | 关闭搜索窗口 |
| `Cmd/Ctrl + 数字` | 快速打开第 N 个结果 |

## 自定义快捷键

你可以为每个插件设置专属快捷键：

1. 打开「设置」→「快捷键」
2. 找到要设置快捷键的插件
3. 点击快捷键输入框
4. 按下想要的组合键
5. 点击「保存」

### 支持的快捷键格式

- `cmd+key` / `ctrl+key` - 修饰键 + 字母/数字
- `cmd+shift+key` - 多修饰键组合
- `f1` - `f12` - 功能键
- `space`, `enter`, `tab` - 特殊按键

### 示例快捷键

```
cmd+1          # Command + 1
ctrl+shift+a   # Ctrl + Shift + A
cmd+shift+5    # Command + Shift + 5
f5             # F5
```

## 快捷键冲突

如果设置的快捷键与其他应用冲突：

1. LTools 会显示警告提示
2. 建议更换为不常用的组合
3. 或禁用其他应用的相同快捷键

## 恢复默认

在快捷键设置页面，点击「恢复默认」可以重置所有快捷键。

## macOS 权限

在 macOS 上，首次使用全局快捷键时需要授予权限：

1. 打开「系统设置」→「隐私与安全性」→「辅助功能」
2. 添加并勾选 LTools
3. 重新启动 LTools
