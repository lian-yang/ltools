# 🖼️ 图床

图片上传和管理工具。

## 功能特性

- ✅ 图片上传
- ✅ 多图床支持
- ✅ 图片管理
- ✅ 快速复制链接
- ✅ 批量上传

## 支持的图床

| 图床 | 免费额度 | 特点 |
|------|----------|------|
| SM.MS | 5 GB | 免费、稳定 |
| Imgur | 无限 | 流行、快速 |
| 腾讯云 COS | 50 GB | 国内快速 |
| 阿里云 OSS | 按量付费 | 稳定可靠 |
| GitHub | 1 GB/仓库 | 开源免费 |
| 自定义 | - | 支持 S3 API |

## 使用方法

### 上传图片

**方式 1：拖拽上传**
1. 打开图床工具
2. 拖拽图片到上传区域
3. 等待上传完成
4. 复制链接

**方式 2：粘贴上传**
1. 复制图片到剪贴板
2. 打开图床工具
3. 自动粘贴并上传

**方式 3：选择文件**
1. 点击「选择图片」
2. 选择图片文件
3. 上传

### 获取链接

上传成功后可以获取：
- **直接链接**：原始图片 URL
- **Markdown**：`![](url)` 格式
- **HTML**：`<img src="url">` 格式
- **BBCode**：`[img]url[/img]` 格式

## 图床配置

### SM.MS

1. 注册 SM.MS 账号
2. 获取 API Token
3. 在设置中配置 Token
4. 开始上传

### GitHub

1. 创建 GitHub 仓库
2. 生成 Personal Access Token
3. 配置仓库和 Token
4. 设置自定义域名（可选）

### 阿里云 OSS

1. 开通阿里云 OSS
2. 创建 Bucket
3. 获取 AccessKey
4. 配置 Bucket 信息

### 自定义图床

支持 S3 API 兼容的图床：
- Endpoint
- Access Key
- Secret Key
- Bucket Name
- Region

## 常见问题

### Q: 上传失败？
**A**:
1. 检查网络连接
2. 确认 API Token 正确
3. 检查图片大小限制
4. 尝试其他图床

### Q: 图片加载慢？
**A**:
1. 使用国内图床
2. 启用 CDN 加速
3. 压缩图片大小

### Q: 链接失效？
**A**:
1. 检查图床状态
2. 备份重要图片
3. 使用多个图床

### Q: 容量不足？
**A**:
1. 清理旧图片
2. 升级图床套餐
3. 使用多个图床

## API 限制

各图床的 API 限制：
- SM.MS：10 次/分钟
- Imgur：1250 次/天
- GitHub：5000 次/小时
