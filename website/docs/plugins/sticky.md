# 📌 贴图工具

将图片置顶显示在屏幕上，方便参考和对比。

## 功能特性

- ✅ 图片置顶显示
- ✅ 自由拖动位置
- ✅ 调整大小和透明度
- ✅ 多图片管理
- ✅ 快捷操作

## 使用方法

### 从截图贴图

1. 使用截图工具截图
2. 在编辑器中按 `Ctrl+T`
3. 截图自动贴图显示

### 从文件贴图

1. 打开全局搜索
2. 输入 `sticky` 或 `贴图`
3. 点击「选择图片」
4. 选择要贴图的图片

### 从剪贴板贴图

1. 复制图片到剪贴板
2. 打开贴图工具
3. 自动检测并贴图

## 贴图操作

### 移动位置

- **拖动**：鼠标拖动贴图窗口
- **快捷键**：方向键微调位置
- **居中**：双击标题栏居中

### 调整大小

- **拖拽边缘**：拖动窗口边缘调整
- **滚轮缩放**：鼠标滚轮缩放
- **精确大小**：右键菜单设置尺寸

### 调整透明度

- **滑块调整**：拖动透明度滑块
- **快捷键**：`Cmd/Ctrl + +/-`
- **范围**：10% - 100%

## 窗口管理

### 多窗口管理

管理多个贴图窗口：
- 窗口列表

### 窗口层级

设置窗口层级：
- **始终置顶**：在所有窗口之上
- **普通**：正常窗口层级
- **仅桌面**：仅在桌面显示

### 排列方式

多个贴图排列：
- 平铺排列
- 层叠排列
- 网格排列
- 自定义排列

## 快捷操作

### 右键菜单

右键点击贴图窗口：
- 复制图片
- 保存图片
- 调整大小
- 设置透明度
- 关闭窗口
- 关闭所有

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Esc` | 关闭贴图 |
| `+` / `-` | 调整透明度 |
| `↑↓←→` | 移动位置 |
| `Cmd/Ctrl+S` | 保存图片 |
| `Cmd/Ctrl+C` | 复制图片 |
| `R` | 旋转图片 |

## 常见问题

### Q: 贴图无法置顶？
**A**:
1. 检查窗口层级设置
2. 某些全屏应用可能覆盖
3. 尝试重新开启置顶

### Q: 图片显示模糊？
**A**:
1. 检查原始图片质量
2. 调整窗口大小
3. 关闭缩放优化

### Q: 占用内存过大？
**A**:
1. 减少贴图数量
2. 使用较小图片
3. 关闭不用的贴图
