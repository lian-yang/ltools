# 📱 二维码

生成和识别二维码的实用工具。

## 功能特性

- ✅ 生成二维码
- ✅ 识别二维码
- ✅ 支持多种内容
- ✅ 自定义样式
- ✅ 保存和分享

## 生成二维码

### 基本使用

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `qrcode` 或 `二维码`
3. 选择「生成二维码」
4. 输入内容
5. 点击生成

### 支持的内容

- **文本**：任意文字内容
- **网址**：网站链接
- **WiFi**：WiFi 连接信息
- **名片**：联系信息
- **邮件**：mailto: 链接
- **电话**：tel: 链接

### 自定义选项

| 选项 | 说明 |
|------|------|
| 尺寸 | 100-500 像素 |
| 颜色 | 前景色和背景色 |
| 容错级别 | L/M/Q/H |
| 边距 | 二维码边距 |

### 保存二维码

- **复制图片**：复制到剪贴板
- **保存文件**：保存为 PNG
- **分享**：直接分享

## 识别二维码

### 从图片识别

1. 选择「识别二维码」
2. 拖放图片文件
3. 或点击选择图片
4. 自动识别并显示内容

### 从剪贴板识别

1. 复制二维码图片
2. 打开插件
3. 自动识别剪贴板中的二维码

### 从摄像头识别

1. 选择「扫描二维码」
2. 授权摄像头权限
3. 对准二维码
4. 自动识别

## 使用场景

- 分享 WiFi 密码
- 生成网址链接
- 名片信息分享
- 活动邀请码
- 支付二维码

## 快捷操作

| 操作 | 说明 |
|------|------|
| 拖放图片 | 直接识别二维码 |
| 复制结果 | 一键复制识别内容 |
| 保存图片 | 保存生成的二维码 |

## 技术细节

- 编码：UTF-8
- 格式：PNG
- 最小尺寸：100x100
- 推荐尺寸：256x256
