# 📋 看板

任务看板管理工具，支持看板（Kanban）方式组织任务。

## 功能特性

- ✅ 看板视图
- ✅ 拖拽排序
- ✅ 多看板支持
- ✅ 标签和优先级
- ✅ 截止日期
- ✅ 本地存储

## 使用方法

### 创建看板

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `kanban` 或 `看板`
3. 点击「新建看板」
4. 输入看板名称

### 看板结构

默认包含三列：
- **待办** (To Do)
- **进行中** (In Progress)
- **已完成** (Done)

可自定义添加更多列。

### 添加任务

1. 点击列底部的「添加卡片」
2. 输入任务标题
3. 可选：添加描述、标签、截止日期
4. 保存

### 任务操作

- **编辑**：点击任务卡片
- **删除**：点击删除图标
- **移动**：拖拽到其他列
- **排序**：在列内拖拽排序

## 任务属性

### 基本信息

| 属性 | 说明 |
|------|------|
| 标题 | 任务名称（必填） |
| 描述 | 详细说明 |
| 标签 | 分类标签（可多个） |
| 优先级 | 高/中/低 |
| 截止日期 | 任务截止时间 |
| 清单 | 子任务列表 |

### 标签系统

创建和管理标签：
- 工作任务 🔵
- 个人事务 🟢
- 重要紧急 🔴
- 自定义标签

### 优先级

- **高** 🔴 - 红色标记
- **中** 🟡 - 黄色标记
- **低** 🔵 - 蓝色标记

## 看板管理

### 多看板

创建多个看板用于不同场景：
- 工作项目
- 个人计划
- 学习任务
- 习惯养成

### 看板设置

- 修改看板名称
- 添加/删除列
- 调整列顺序
- 设置 WIP 限制

## 数据存储

### 本地存储

- 数据保存在本地
- 自动保存更改
- 支持导出备份

### 导出功能

1. 点击看板菜单
2. 选择「导出」
3. 保存为 JSON 文件

### 导入功能

1. 点击「导入看板」
2. 选择 JSON 文件
3. 确认导入

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `N` | 新建任务 |
| `E` | 编辑选中任务 |
| `Delete` | 删除任务 |
| `↑` `↓` `←` `→` | 导航 |
| `Enter` | 打开任务详情 |
| `Esc` | 关闭详情 |

## 使用场景

### 项目管理

```
待办 → 进行中 → 测试 → 已完成
```

### 个人计划

```
本周 → 本月 → 长期 → 已完成
```

### 学习计划

```
要学 → 学习中 → 复习 → 已掌握
```

## 最佳实践

1. **保持简单**：不要创建太多列
2. **定期回顾**：每周回顾和清理
3. **WIP 限制**：限制每列任务数量
4. **优先级**：合理设置任务优先级
5. **子任务**：大任务拆分为子任务
