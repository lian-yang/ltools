# 📄 Markdown

Markdown 编辑和预览工具，支持实时渲染。

## 功能特性

- ✅ Markdown 编辑
- ✅ 实时预览
- ✅ 语法高亮
- ✅ 导出功能
- ✅ 多种主题

## 使用方法

### 创建文档

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `markdown` 或 `md`
3. 开始编辑 Markdown

### 编辑模式

**分屏模式**：
- 左侧：编辑器
- 右侧：实时预览
- 同步滚动

**纯编辑模式**：
- 全屏编辑
- 专注写作

**纯预览模式**：
- 只显示预览
- 查看最终效果

## Markdown 语法

### 基本语法

```markdown
# 标题 1
## 标题 2
### 标题 3

**粗体** *斜体* ~~删除线~~

- 无序列表 1
- 无序列表 2

1. 有序列表 1
2. 有序列表 2

[链接](https://example.com)
![图片](image.png)

`行内代码`

\`\`\`
代码块
\`\`\`

> 引用

---
分隔线
```

### 扩展语法

**表格**：
```markdown
| 列1 | 列2 | 列3 |
|-----|-----|-----|
| 内容 | 内容 | 内容 |
```

**任务列表**：
```markdown
- [x] 已完成
- [ ] 未完成
```

**脚注**：
```markdown
这是一个脚注[^1]

[^1]: 脚注内容
```

## 导出功能

### 导出格式

- **HTML**：带样式的网页
- **PDF**：PDF 文档

### 导出步骤

1. 点击「导出」按钮
2. 选择导出格式
3. 设置导出选项
4. 选择保存位置

### 导出选项

- 内联样式

- 包含目录

**PDF 导出**：
- 页面大小（A4/Letter）
- 边距设置
- 横向/纵向

## 常见问题

### Q: 预览不显示？
**A**: 检查是否切换到纯编辑模式，切换回分屏模式。

### Q: 导出 PDF 失败？
**A**: 确保系统支持 PDF 导出，检查磁盘空间。

### Q: 语法高亮不正确？
**A**: 检查代码块是否指定了语言，刷新预览。
