# 🔒 密码库

本地加密存储密码和敏感信息的安全保险箱。

## 功能特性

- ✅ 本地加密存储
- ✅ 主密码保护
- ✅ 分类管理
- ✅ 快速搜索
- ✅ 自动填充（计划中）

## 首次使用

### 创建密码库

1. 打开插件
2. 设置主密码（请牢记！）
3. 确认主密码
4. 开始添加条目

> ⚠️ **重要**：主密码无法找回，请务必牢记！

## 使用方法

### 解锁密码库

1. 打开全局搜索
2. 输入 `vault` 或 `密码库`
3. 输入主密码
4. 访问存储的密码

### 添加条目

点击「添加」按钮，填写信息：

**登录信息**：
- 标题（如：GitHub）
- 用户名/邮箱
- 密码
- 网址
- 备注

**支持类型**：
- 登录凭据
- 信用卡信息
- 安全备注
- WiFi 密码
- 软件许可证

### 分类管理

创建分类组织密码：
- 工作
- 个人
- 财务
- 社交
- 自定义分类

## 安全特性

### 加密算法

- AES-256 加密
- 主密码使用 PBKDF2 派生
- 本地存储，不上传云端

### 安全建议

- ✅ 使用强主密码
- ✅ 定期更换主密码
- ✅ 不要共享主密码
- ✅ 定期备份密码库

## 数据管理

### 备份

1. 点击「导出」
2. 选择导出位置
3. 输入主密码确认
4. 保存加密备份文件

### 恢复

1. 点击「导入」
2. 选择备份文件
3. 输入原主密码
4. 确认导入

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl+N` | 新建条目 |
| `Cmd/Ctrl+F` | 搜索 |
| `Cmd/Ctrl+C` | 复制密码 |
| `Esc` | 锁定密码库 |

## 常见问题

**Q: 忘记主密码怎么办？**
A: 主密码无法找回。建议定期备份并安全存储主密码。

**Q: 数据存储在哪里？**
A: 所有数据加密存储在本地 `~/.ltools/vault/` 目录。

**Q: 可以同步到其他设备吗？**
A: 目前需要手动导入/导出。云同步功能计划中。

**Q: 更换电脑怎么办？**
A: 导出备份文件，在新电脑上导入即可。
