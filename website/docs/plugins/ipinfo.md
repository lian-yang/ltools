# 🌍 IP 信息

查询 IP 地址的归属信息和地理位置。

## 功能特性

- ✅ 查询本机 IP
- ✅ 查询任意 IP
- ✅ 地理位置定位
- ✅ ISP 信息
- ✅ 历史记录

## 使用方法

### 查询本机 IP

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `ip` 或 `IP`
3. 自动显示本机公网 IP

### 查询指定 IP

1. 在搜索框输入 IP 地址
2. 按回车查询
3. 查看详细信息

### 查询历史

- 查看最近查询的 IP
- 快速重新查询
- 清除历史记录

## 查询结果

## 常见问题

### Q: 位置信息不准确？
**A**: IP 定位基于数据库，城市级别通常准确，街道级别可能有偏差。

### Q: 查询失败？
**A**:
1. 检查网络连接
2. 确认 API 服务可用
3. 稍后重试

### Q: 显示的是内网 IP？
**A**: 本机 IP 显示的是公网 IP，如显示内网 IP 请检查网络配置。
