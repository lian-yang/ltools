# 🔖 书签搜索

快速搜索浏览器书签，支持 Chrome、Firefox 和 Safari。

## 功能特性

- ✅ 多浏览器支持
- ✅ 快速搜索
- ✅ 书签同步
- ✅ 分类浏览
- ✅ 一键打开

## 支持的浏览器

| 浏览器 | macOS | Windows | Linux |
|--------|-------|---------|-------|
| Chrome | ✅ | ✅ | ✅ |
| Firefox | ✅ | ✅ | ✅ |
| Safari | ✅ | ❌ | ❌ |
| Edge | ✅ | ✅ | ✅ |

## 使用方法

### 搜索书签

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `bookmark` 或 `书签`
3. 输入搜索关键词
4. 从结果中选择
5. 回车打开网址

### 搜索范围

搜索书签的：
- **标题**：书签名称
- **网址**：URL 地址
- **文件夹**：所在文件夹
- **描述**：书签描述

### 搜索技巧

- **域名搜索**：输入 `github.com`
- **标题搜索**：输入书签标题关键词
- **模糊匹配**：部分匹配即可
- **拼音搜索**：支持中文拼音

## 书签同步

### 自动同步

首次打开时自动导入书签：
1. 扫描浏览器书签
2. 建立索引
3. 缓存到本地

### 手动同步

1. 点击「同步书签」
2. 选择浏览器
3. 等待同步完成

### 同步频率

- 自动：每天一次
- 手动：随时同步
- 设置：可配置频率

## 数据存储

### 本地缓存

- 书签数据缓存在本地
- 加密存储敏感信息
- 可随时清除缓存

### 缓存位置

- **macOS**: `~/.ltools/bookmarks/`
- **Windows**: `%USERPROFILE%\.ltools\bookmarks\`
- **Linux**: `~/.ltools/bookmarks/`

## 隐私说明

- ✅ 仅读取书签数据
- ✅ 不上传到服务器
- ✅ 本地加密存储
- ✅ 不修改浏览器书签

## 常见问题

### Q: 书签没有同步？
**A**:
1. 检查浏览器是否安装
2. 手动点击同步
3. 检查浏览器是否运行

### Q: 搜索不到书签？
**A**:
1. 刷新书签索引
2. 清除缓存重新同步
3. 检查关键词是否正确

### Q: 书签打开失败？
**A**:
1. 检查网址是否有效
2. 确认默认浏览器设置
3. 尝试其他浏览器打开
