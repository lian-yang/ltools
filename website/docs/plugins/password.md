# 🔐 密码生成器

生成安全的随机密码，支持自定义密码规则。

## 功能特性

- ✅ 自定义密码长度
- ✅ 选择字符类型
- ✅ 排除相似字符
- ✅ 一键复制
- ✅ 批量生成

## 使用方法

### 生成密码

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `password` 或 `密码`
3. 配置密码规则
4. 点击生成按钮

### 配置选项

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 长度 | 密码长度 | 16 |
| 大写字母 | A-Z | ✅ |
| 小写字母 | a-z | ✅ |
| 数字 | 0-9 | ✅ |
| 特殊字符 | !@#$%^&* | ✅ |
| 排除相似 | 0/O, 1/l/I | ❌ |

### 预设配置

- **简单**：仅数字和字母，8位
- **中等**：包含特殊字符，12位（推荐）
- **强**：所有字符类型，16位
- **极强**：所有字符类型，24位

## 安全性说明

### 密码强度

- 使用加密随机数生成器
- 每次生成都完全随机
- 不保存生成的密码

### 最佳实践

- ✅ 密码长度 >= 12 位
- ✅ 包含多种字符类型
- ✅ 不同网站使用不同密码
- ✅ 使用密码管理器保存

## 快捷操作

- **生成**：点击生成按钮或按 `Enter`
- **复制**：点击密码自动复制
- **刷新**：再次生成新密码
- **批量**：一次生成多个密码

## 使用场景

- 注册新账号
- 定期更换密码
- 临时密码生成
- API 密钥生成
