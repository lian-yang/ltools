# 🎵 音乐播放器

内置音源服务的音乐播放器，支持多音源、歌词同步、播放列表管理。

## 功能特性

- ✅ 多音源支持（可配置）
- ✅ 歌词同步显示
- ✅ 播放列表管理
- ✅ 本地缓存
- ✅ 搜索歌曲
- ✅ 收藏歌曲

## 使用方法

### 首次使用

首次打开音乐播放器时，需要初始化音源服务：

1. 打开音乐播放器插件
2. 等待音源服务自动配置
3. 配置完成后即可使用

::: warning 系统要求
音乐播放器需要 **Node.js >= 16.0.0**
:::

### 搜索歌曲

1. 在搜索框输入歌曲名或歌手
2. 按回车搜索
3. 点击结果播放

### 播放控制

- **播放/暂停**: 点击播放按钮或按空格键
- **上一首/下一首**: 点击切换按钮
- **进度跳转**: 拖动进度条
- **音量调节**: 拖动音量滑块

### 歌词显示

- 自动加载歌词
- 同步高亮当前歌词
- 点击歌词跳转

### 播放列表

- 添加歌曲到播放列表
- 管理播放顺序
- 清空播放列表

## 音源配置

### 添加音源

1. 打开设置
2. 选择「音乐播放器」
3. 添加音源 URL
4. 测试并保存

### 音源切换

支持多个音源，可在设置中切换。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 播放/暂停 |
| `←` | 上一首 |
| `→` | 下一首 |
| `↑` | 增加音量 |
| `↓` | 减少音量 |

## 数据存储

- 播放列表保存在本地
- 歌曲缓存可配置
- 收藏信息本地存储

## 注意事项

- 音源服务需要网络连接
- 部分歌曲可能需要切换音源
- 缓存文件可定期清理
