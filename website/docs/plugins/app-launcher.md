# 🚀 应用启动器

快速启动应用程序，支持搜索和快捷键。

## 功能特性

- ✅ 快速搜索应用
- ✅ 一键启动
- ✅ 应用图标显示
- ✅ 最近使用记录
- ✅ 自定义快捷键

## 使用方法

### 搜索应用

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入应用名称或关键词
3. 从搜索结果选择应用
4. 回车或点击启动

### 搜索技巧

- **模糊搜索**：输入部分名称即可
- **拼音搜索**：支持中文拼音
- **缩写搜索**：如 `chrome` → `chr`
- **关键词**：如 `浏览器` → Chrome

### 最近使用

搜索窗口显示：
- 最近使用的应用
- 使用频率统计
- 快速访问

## 应用管理

### 应用列表

自动扫描系统应用：
- **macOS**：`/Applications/`、`~/Applications/`
- **Windows**：开始菜单、桌面
- **Linux**：`.desktop` 文件

### 刷新缓存

1. 打开设置
2. 点击「刷新应用列表」
3. 等待扫描完成

### 应用信息

查看应用详情：
- 应用名称
- 安装路径
- 版本信息
- 最后使用时间

## 快捷键

### 全局搜索

- `Cmd+5` (macOS)
- `Ctrl+5` (Windows/Linux)

### 应用快捷键

为常用应用设置专属快捷键：

1. 打开「设置」→「快捷键」
2. 找到应用启动器
3. 添加应用快捷键
4. 保存配置

**示例**：
- `Cmd+Shift+C` → Chrome
- `Cmd+Shift+V` → VS Code
- `Cmd+Shift+T` → Terminal

## 常见问题

### Q: 找不到某个应用？
**A**: 点击「刷新应用列表」重新扫描

### Q: 应用图标不显示？
**A**: 检查应用是否正确安装，刷新缓存

### Q: 搜索速度慢？
**A**: 减少索引的应用数量，清除历史记录
