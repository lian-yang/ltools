# ⚙️ 进程管理

查看和管理系统进程的工具。

## 功能特性

- ✅ 进程列表
- ✅ 资源使用显示
- ✅ 进程搜索
- ✅ 强制结束进程
- ✅ 进程详情

## 使用方法

### 查看进程

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `process` 或 `进程`
3. 查看所有运行中的进程

### 进程列表

显示信息包括：
- **进程名称**：程序名称
- **PID**：进程 ID
- **CPU%**：CPU 使用率
- **内存%**：内存使用率
- **状态**：运行状态
- **用户**：运行用户

### 排序功能

点击列标题排序：
- 按 CPU 使用率排序
- 按内存使用排序
- 按进程名排序
- 按 PID 排序

## 进程操作

### 查看详情

点击进程查看详细信息：
- 完整命令行
- 启动时间
- 运行时长
- 父进程
- 子进程列表
- 打开的文件
- 网络连接

### 结束进程

**普通结束**：
1. 选中进程
2. 点击「结束进程」
3. 确认操作

**强制结束**：
1. 选中进程
2. 点击「强制结束」
3. 确认操作

> ⚠️ **注意**：强制结束可能导致数据丢失

### 批量操作

1. 勾选多个进程
2. 选择批量操作
3. 批量结束/暂停

## 搜索过滤

### 快速搜索

在搜索框输入：
- 进程名称
- PID
- 用户名
- 命令行关键字

### 高级过滤

- 按用户过滤
- 按状态过滤（运行/睡眠/停止）
- 按资源使用过滤（CPU > X%）
- 组合过滤条件

## 资源监控

### Top 进程

自动显示资源占用最高的进程：
- **CPU Top 10**：CPU 占用最高
- **内存 Top 10**：内存占用最高

### 实时更新

- 自动刷新进程列表
- 可配置刷新间隔
- 暂停/恢复刷新

## 进程信息

### 状态说明

| 状态 | 说明 |
|------|------|
| Running | 正在运行 |
| Sleeping | 休眠中 |
| Stopped | 已停止 |
| Zombie | 僵尸进程 |
| Idle | 空闲 |

### 优先级

查看和修改进程优先级：
- Nice 值（-20 到 19）
- 实时优先级
- I/O 优先级

## 权限说明

### macOS/Linux

- 普通用户：只能结束自己的进程
- 管理员：可以结束所有进程
- 需要 sudo 权限结束系统进程

### Windows

- 普通用户：只能结束自己的进程
- 管理员：可以结束大部分进程
- 系统进程：需要特殊权限

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `F5` | 刷新列表 |
| `Delete` | 结束选中进程 |
| `Shift+Delete` | 强制结束 |
| `Ctrl+F` | 搜索进程 |
| `Esc` | 清除选择 |

## 使用场景

### 性能优化

- 找出占用 CPU/内存的进程
- 结束无响应的程序
- 监控系统资源使用

### 故障排查

- 查找异常进程
- 检测恶意程序
- 分析系统卡顿原因

### 系统维护

- 清理僵尸进程
- 管理后台服务
- 控制进程优先级

## 安全提示

- ⚠️ 不要随意结束系统进程
- ⚠️ 结束进程前确认其用途
- ⚠️ 重要工作先保存
- ⚠️ 无法结束的进程不要强制

## 技术细节

### 数据来源

- **macOS**：ps 命令、sysctl
- **Windows**：WMI、Performance Counters
- **Linux**：/proc 文件系统

### 更新频率

- 默认：2 秒
- 可配置：1-10 秒
- 自动刷新可关闭
