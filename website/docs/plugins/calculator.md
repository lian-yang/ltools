# 🔢 计算器

支持基本四则运算和表达式求值的计算器插件。

## 功能特性

- ✅ 基本运算（加减乘除）
- ✅ 表达式求值
- ✅ 百分比计算
- ✅ 历史记录
- ✅ 快速复制结果

## 使用方法

### 基本运算

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `calc` 或 `计算`
3. 输入算式，如 `123 + 456`
4. 自动计算并显示结果

### 支持的运算

| 运算符 | 说明 | 示例 |
|--------|------|------|
| `+` | 加法 | `10 + 20` = 30 |
| `-` | 减法 | `50 - 20` = 30 |
| `*` | 乘法 | `5 * 6` = 30 |
| `/` | 除法 | `60 / 2` = 30 |
| `%` | 百分比 | `100 * 20%` = 20 |
| `()` | 括号 | `(10 + 5) * 2` = 30 |

### 表达式示例

```
123 + 456          # 基本加法
100 * 20%          # 百分比计算
(50 + 30) * 2      # 带括号的运算
150 / 3 + 25       # 混合运算
```

## 历史记录

- 自动保存计算历史
- 点击历史记录快速复制结果
- 清空历史记录功能

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 复制结果 |
| `Esc` | 清空输入 |
| `↑` / `↓` | 浏览历史 |

## 使用技巧

- 直接输入算式即可自动计算
- 支持 `Ctrl+C` 复制结果
- 历史记录会保存最近 50 条计算
