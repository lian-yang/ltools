# 📝 JSON 编辑器

JSON 编辑器提供专业的 JSON 格式化、验证和编辑功能，基于 Monaco Editor（VS Code 同款编辑器）。

## 功能特性

- ✅ JSON 格式化/压缩
- ✅ 语法验证和高亮
- ✅ 树形结构查看
- ✅ 搜索和替换
- ✅ 代码折叠
- ✅ 多主题支持
- ✅ 大文件支持

## 使用方法

### 打开编辑器

1. 打开全局搜索 (`Cmd/Ctrl+5`)
2. 输入 `json` 或 `JSON`
3. 回车打开

### 基本操作

**格式化**: `Cmd/Ctrl+Shift+F`
**压缩**: `Cmd/Ctrl+Shift+M`
**验证**: 实时显示语法错误
**复制**: `Cmd/Ctrl+C`
**粘贴**: `Cmd/Ctrl+V`

### 编辑器功能

- **语法高亮**: 关键字、字符串、数字不同颜色
- **自动补全**: 智能提示和补全
- **错误提示**: 红线标记语法错误
- **迷你地图**: 右侧代码概览

## 导入导出

### 导入 JSON

- 直接粘贴 JSON 内容
- 拖放 JSON 文件到编辑器

### 导出 JSON

- 复制到剪贴板
- 保存为文件

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Cmd/Ctrl+F` | 查找 |
| `Cmd/Ctrl+H` | 替换 |
| `Cmd/Ctrl+Shift+F` | 格式化 |
| `Cmd/Ctrl+Shift+M` | 压缩 |
| `F1` | 命令面板 |

## 配置

### 主题

支持多种编辑器主题：
- VS Code Dark (默认)
- VS Code Light
- Monokai
- GitHub

### 字体大小

可调整编辑器字体大小。

## 技巧

- 按 `Ctrl+Space` 触发自动补全
- 按 `F12` 跳转到定义
- 使用折叠功能处理大 JSON
- 错误提示会显示具体位置和原因
